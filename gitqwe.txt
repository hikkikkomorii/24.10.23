1 - git init
инициализирует новый репозиторий Git в текущей папке
2 - git add .
добавляет изменения в определенный файл в индекс
3 - git commit -m "first commit"
фиксирует изменения в репозитории с комментарием
4 - git branch -M main
управляет ветками в репозитории.
5 - git remote add origin "здесь ссылка на твой репозиторий"
служит для управления списком удалённых репозиториев
6 - git push -u origin main
отправляет изменения в удаленный репозиторий
